About this repository

Use this landing page to route people to your full collection, while keeping a neat preview and citation-ready metadata.

Recommended repo structure

Keep the landing page + data clean and predictable:

  • /assets/ specimen photos (web-optimized + originals if needed)
  • /data/specimens.csv or /data/specimens.json as the source of truth
  • /docs/ if you prefer publishing from docs via GitHub Pages
  • README.md with a one-paragraph mission + how to cite

How to scale this (nicely)

When your catalog grows, switch the preview to load from a JSON file in /data. You can also add:

  • Map view (localities) via GeoJSON.
  • QR per specimen (links to #FV-TRI-001 anchors).
  • DOI via Zenodo for citability.
  • Minimal taxonomy/stratigraphy controlled vocabularies.
